Una vez abierto el servidor de Mongo, debemos conectarnos a el para poder realizar consultas.

El shell de Mongo es un interprete que nos permite ejecutar comandos directamente en la
base de datos, y realizar cambios y consultas.

Para ejecutar el shell de Mongo, hay que ejecutar el comando:

> mongosh





Para saber si se conectó correctamente a la base de datos, nota la entrada de texto, deberia decir:

test>



¡Has accedido a la base de datos!
